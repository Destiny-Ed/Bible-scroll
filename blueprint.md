
# BibleScroll App Blueprint

## Overview

BibleScroll is a mobile application that provides a TikTok-like vertical scrolling experience for watching short, engaging videos based on Bible stories. The app also includes features for searching, daily reading plans, bookmarking, and user profiles.

## Style, Design, and Features

### Implemented

*   **Theme:** Light and dark mode with a modern, clean aesthetic using the Lato font from Google Fonts.
*   **Navigation:** Bottom navigation bar for easy access to Home (Feed), Search, Daily Reading Plan, Bookmarks, and Profile.
*   **Feed:** Vertical scrolling feed of video content with a `VideoCard` widget that will play videos.
*   **Search:** A basic search screen with a search bar.
*   **Daily Reading Plan:** A screen to display a list of reading plans.
*   **Bookmarks:** A screen to display a grid of bookmarked videos.
*   **Profile:** A user profile screen with a header, stats, and a tabbed view for user's videos and liked videos.
